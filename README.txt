Le dossier "Quantification brute" est une étude préliminaire à l'implémentation de l'algorithme LBG traditionnels.

Le script principal est LBG.py, le dictionnaire généré par ce script peut être enregistré (avec pickle) dans le fichier .txt "dictionary1024.txt".

La base de données utilisée est présentée en 1024x1024 (large), et la base réduite en 512x512 (small).
